# Get and set today's wallpaper from Bing and momentum

## Installation

```bash
$ npm install -g picday-cli
```

## How to use

- change wallpaper to Bing

  ```bash
  $ picday use bing
  ```

- change wallpaper to momentum

  ```bash
  $ picday use momentum
  ```

- reset origin wallpaper

  ```bash
  $ picday reset
  ```

- delete today's wallpaper

  ```bash
  $ picday clean
  ```
